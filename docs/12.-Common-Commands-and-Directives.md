# 5. Liquibase Changeset Directions

## Environment Variables

Use \${VAR_NAME} syntax

<div class="code panel pdl conf-macro output-block" hasbody="true" macro-id="460b77cb-a739-40d8-82e1-333261642e14" macro-name="code" style="border-width: 1px;">

<div class="codeContent panelContent pdl">

``` sql
begin
   cla_apex.cla_configuration_pkg.upsert (
        p_instance_id =>1 ,
        p_key_type_id => 1,
        p_key         => 'ams_server',
        p_description => 'Local AMS (APEX Messaging Service) server URL',
        p_value       => 'https://${AMS_SERVER_FQDN}',
        p_key_group   => 'ams'
    );  
end;
/
```

</div>
